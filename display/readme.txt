Block
=====

A block element occupies the entire space of its parent element (container).

A block element always starts on a new line and takes up the full width available.

    - p, h1, ..., h6
    - ol, ul
    - div
    - form
    - hr
    - table


inline
=======

Inline elements can start anywhere in a line.

    - i, em, strong, a
    - br, img, script, span
    - button, input, label, select, textarea
    - nav


    - respect left and right margins and padding, but not top and bottom.

    - no les afecta el width. Para usar width hay q convertirlos en inline-block elements.

==========

    - li


    - soporta heights/widths, margin, padding, float
